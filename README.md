# 🏏 IPL Match Winning Prediction

Predicting the winning probability of IPL teams during an ongoing match using machine learning.

---

## 📌 Overview

Predicting the outcome of a cricket match is a challenging classification problem due to the dynamic nature of the game. This project uses historical IPL match data to estimate the probability of a team winning during an ongoing match based on current match conditions.

The project compares multiple machine learning models to identify an effective approach for predicting match outcomes.

---

## 🎯 Objective

The objective of this project is to:

- Predict the winning probability of IPL teams during live matches.
- Compare machine learning algorithms for match outcome prediction.
- Evaluate model performance using historical IPL data.

---

## 📊 Dataset

**Source:** IPL Match and Ball-by-Ball Dataset (2008–2019)

The dataset includes:

- Match information
- Ball-by-ball deliveries
- Team details
- Venue information
- Match results

### Features

- Batting Team
- Bowling Team
- City
- Target Score
- Current Score
- Overs Completed
- Current Run Rate
- Required Run Rate
